A simple but complete developer toolkit for UWP developers. It provides users with following functionality:

- Icon Browser
- URI Tester
- Assets Generator
- Windows Blog
- GUID Generator
- Regular Expression Tester
- Json2C#
- Color Palette
- Regex Helper

Note: Some features will be coming in future release.
